# Matika

A Czech game to practice maths.

## Constraints

- runs in browser, static artifact (either a plain file or at least a static compilation output)
- no remote communication
- browser local storage
- all texts are in Czech language

## Theme

Minecraft - blocks, pixelated graphics, addons to Minecraft character as rewards

## Users

At the beginning, the user selects a character, the character's progress and preferences is
preserved.
